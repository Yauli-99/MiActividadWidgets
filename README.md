# MiActividadWidgets

Aplicación Android desarrollada en Android Studio utilizando Java y XML.

## Widgets utilizados
- TextView
- EditText
- RadioButton
- CheckBox
- Button

## Funcionalidades
- Registro de nombre
- Selección de curso
- Validación de términos
- Mensajes Toast

## Herramientas
- Android Studio
- Java
- XML
