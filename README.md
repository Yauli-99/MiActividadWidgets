# MiActividadWidgets

Aplicación móvil desarrollada en Android Studio utilizando Java y XML como parte de la actividad del curso Desarrollo de Aplicaciones Móviles.

## Descripción del Proyecto

La aplicación fue desarrollada con el objetivo de comprender el funcionamiento básico de Android Studio, el uso de Activities, interfaces gráficas mediante XML y la interacción entre widgets.

El proyecto permite al usuario:
- Ingresar su nombre
- Seleccionar un curso
- Aceptar términos y condiciones
- Mostrar información mediante mensajes Toast

## Tecnologías Utilizadas

- Android Studio
- Java
- XML
- Android SDK

## Widgets Implementados

La aplicación utiliza los siguientes componentes gráficos:

- TextView
- EditText
- RadioButton
- CheckBox
- Button

## Funcionalidades

- Creación de interfaz gráfica con XML
- Programación de eventos con Java
- Uso del método onCreate()
- Interacción entre componentes
- Validación de selección mediante CheckBox
- Visualización de mensajes Toast

## Estructura del Proyecto

El proyecto contiene los siguientes archivos principales:

- MainActivity.java
- activity_main.xml

## Ejecución de la Aplicación

La aplicación fue ejecutada correctamente en el emulador/dispositivo móvil verificando el correcto funcionamiento de todos los componentes implementados.
